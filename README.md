# Smart Lead Automation

This is a full-stack Smart Lead Automation project built as a company assignment.

## Tech Stack
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Frontend: React
- Scheduler: Cron Job

## Features
- REST API to receive lead data
- MongoDB integration using Mongoose
- Cron job runs every 5 minutes to sync leads
- Simple React UI to send data to backend
